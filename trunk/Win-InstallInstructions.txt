Windows build instructions:
---------------------------

The author creates windows builds by means of cross-compiling the application
from Linux with mingw-w64

Requires:

Linux (The author uses Ubuntu 11.10) with usual development tools
mingw-w64
wxWidgets 
	(The author used wxGTK-2.8.12 cross-compiled like this:
	1. In the extracted wx source folder a new win-ur-static folder was created
	2. cd into the new folder and run:
../configure --host=i686-w64-mingw32 --prefix=/PATH_TO_WX_SOURCE/win-ur-static/inst --enable-unicode --disable-shared
	3. make
	4. make install
)


Compile libsndfile:
in lib-src/libsndfile folder
	./configure --host=i686-w64-mingw32 --disable-external-libs
	make
in lib-src folder
	ln -sf libsndfile/src/.libs/libsndfile.a .


Compile RtAudio:
in lib-src/rtaudio folder
	./configure --host=i686-w64-mingw32 --with-ds
	make
	i686-w64-mingw32-ranlib librtaudio.a
in lib-src folder
	ln -sf rtaudio/librtaudio.a .



In src folder:

i686-w64-mingw32-g++ -D__WINDOWS_DS__ -static -O2 -mmmx -msse -msse2 -msse3 -mssse3 -mfpmath=sse -o LoopAuditioneer.exe LoopAuditioneer.cpp MyFrame.cpp MyPanel.cpp CueMarkers.cpp LoopMarkers.cpp FileHandling.cpp MySound.cpp WaveformDrawer.cpp LoopParametersDialog.cpp BatchProcessDialog.cpp AutoLoopDialog.cpp AutoLooping.cpp PitchDialog.cpp FFT.cpp -I../lib-src/libsndfile/src -I../lib-src/rtaudio -I../lib-src/rtaudio/include ../lib-src/libsndfile.a ../lib-src/librtaudio.a -ldsound -lole32 -lwinmm -lm `/PATH_TO_WX_SOURCE/win-ur-static/wx-config --static=yes --cxxflags --libs`


To reduce size of binary:

strip --strip-all LoopAuditioneer.exe

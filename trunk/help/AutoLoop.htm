Automated loopsearch

The automatic search for loops that can be done by LoopAuditioneer is highly configurable by the user to ensure that good results can be achieved. However, to get the most out of this feature some knowledge about how it works might be necessary.

The search for loops is controlled by the loopsearch settings that decide the parameters for the search algorithm. Some adjustments of the settings will almoast always be necessary to get the best results for different samples.

The loops that are found are all natural loop points and no modification of the audio data takes place. The information is stored in the .wav file metadata (smpl chunk).

Menu bar

The menu bar has five menus:

  • File: file operations, loopsearch settings and exit
  • View: zoom options
  • Transport: play, stop, loop play option
  • Tools: various things to do with an open file
  • Help: help and about

File menu

The file menu contains commands for choosing working directory, opening and saving files, adjusting loopsearch settings and exiting the program.

Choose folder

This item open a file chooser dialog where the current working directory is selected. When a folder is chosen the file list will be populated with all .wav files in that directory.

Open file

This command opens the currently selected file in the file list. The command can also be used to "undo" any work on the file and reload it from disk.

Save

The currently open file will be saved and overwrite the file on disk.

Save as...

This command opens a dialog that gives the user a choice to save the file with another name and in another folder if that is desired.

Save, open next

Selecting this command saves the currently open file and proceeds to open next file in the file list thus allowing swifter work when processing many files.

Autoloop settings

This item opens up the settings dialog where all parameters for automatic loopsearch can be adjusted.

Exit

This command immediately shuts the program down without saving currently opened file or asking the user to save.

View menu

The view menu has options to zoom in and out on the waveform in amplitude (not in time).

Zoom in

Every time this option is selected the waveform will be scaled twice as large (in height). The maximum zoom level is 64 times the natural size scaling.

Zoom out

This command reduces the zoom level by half, thus making the waveform smaller. The lowest zoom level is the natural 1:1 size.

Transport menu

In the transport menu the options to control playback of the selected loop or cue of an opened file can be found.

Start

Selecting this item starts playback of the selected loop or cue. A loop or cue must exist and be selected or else playback won't be possible.

Stop

This item stops the playback of the selected loop or cue.

Loop only

If this menu item is selected the playback won't start at the beginning of the file. Instead it will start just before the end of the loop. If it's de-selected again the playback will be from the beginning of the file.

Tools menu

In the tools menu most of the operations that can be done on a file can be found. Also the batch dialog can be invoked from this menu.

New loop

This opens up the loop parameter dialog where the user can set the start and end of a new loop. The values are given in samples and can be written into the value boxes or stepped up/down with the spinner arrows.

Autoloop

When selected this item starts an automatic search of loops in the opened file with the currently used loopsearch settings. Depending on the settings this can take some time until completed, have patience.

Batch processing

This is the only tool that always is available for usage, even if no file is opened. When selected the batch process dialog will be opened where different operations can be done on all .wav files in a folder.

Pitch detection

When selected this item will attempt a pitch detection on the currently opened file with three different methods. It's also possible to manually set the pitch information in the dialog that opens.

Crossfade

The crossfade option affects the currently selected loop and opens up a dialog in which different methods and lengths of the crossfade can be chosen.

Cut and fade

The cut and fade dialog that opens when this item is selected makes it possible to trim start and end of a file as well as performing fades in and out. Every operation can be done independently or all at once.

Help menu

The help menu contans this help as well as an option to display a little information about this program.

Help

When selected this help will be displayed in it's own window.

About

This item opens the about dialog that display some brief information about this program.
